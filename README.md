# Money

[![Build and deploy](https://github.com/mattiassamskar/Money/actions/workflows/build-and-deploy.yml/badge.svg)](https://github.com/mattiassamskar/Money/actions/workflows/build-and-deploy.yml)

## Develop

### Backend

`dotnet run` in Money.Web project starts the API on port 5000.

`MockDbService` can be used as in-memory mock database.

Configure `MoneyDbConnectionString` environment variable to use prod database.

### Frontend

`npm run dev` in client folder starts dev server.
